## 项目介绍
该项目是一个完整的移动端浏览器指纹检测项目，包括但不仅限于：

* 系统相关 
* 版本信息 
* 字体相关 
* Navigator 相关
* Client Hints
* 性能相关
* 运行内存相关
* 窗口相关
* 媒体相关
* Screen 相关
* Rect 相关
* Document 相关
* Header 相关
* Cookie 相关
* 电量相关
* 网络相关
* 指纹相关
* 显卡相关
* 重力感应相关
* WebRTC相关
* 位置相关

## 启动服务

### 拉取项目

``git clone https://github.com/tyua07/FP-Browser-Detect.git``

### 设置镜像

``npm config set registry https://registry.npm.taobao.org``

### 安装 依赖

* ``cd FP-Browser-Detect``
* ``npm install``

### 启动服务

``npm run start``
